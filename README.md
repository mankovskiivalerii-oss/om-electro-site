# ООО «Электро-монтажная компания “ОМ”» — лендинг

Одностраничный сайт на React + Vite + Tailwind CSS для компании, выполняющей электромонтажные и слаботочные работы в Санкт-Петербурге и Ленинградской области.

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
npm run preview
```

## Стек

- React
- Vite
- Tailwind CSS
- lucide-react
